# 05 Copywriting

## 1. Purpose and Scope
This document defines production-ready website copy for Dama Venus, aligned with the approved project brief, creative direction, asset strategy, homepage concept, and current status.

Scope:
- Homepage-first copy system with direct component usability.
- Reusable copy for core sections: Navigation, Intro/Lead, Release, Visuals, About, Services, Press/EPK, Contact, Footer.
- Option sets per component, each with one clear recommendation.

Not in scope:
- Long-form biography texts.
- Technical implementation details.
- Placeholder/lorem copy.

---

## 2. Voice Framework (Binding)
Core tone: cinematic, dark, feminine, elegant, artsy, modern, mystical, editorial, high-end.

Voice principles:
1. Short lines, high precision, no filler.
2. Sensory + visual language over promotional claims.
3. Controlled intensity: calm, magnetic, never loud.
4. Specific verbs, minimal adjectives.
5. No generic music-PR phrases.

Do:
- Build atmosphere in one line.
- Lead toward action without sounding salesy.

Don’t:
- “Welcome to my official website.”
- “Unique blend of sounds.”
- “Passionate artist pushing boundaries.”

---

## 3. Structural Bilingual Readiness (EN-first, DE-ready)
Copy is written in compact semantic blocks to simplify future DE adaptation:
- Headline (2–6 words)
- Subhead (6–14 words)
- Optional context line (max. 12 words)
- CTA (1–3 words)

DE transfer guidance:
- Preserve rhythm before literal wording.
- Keep sentence length similar for layout stability.
- Prefer concrete nouns/verbs; avoid compounded long DE clauses.
