# 00 Project Brief

## 1. Projektname
Dama Venus Website

## 2. Projektziel
Entwicklung einer hochwertigen, bildstarken Artist-Website für Dama Venus mit klarer Premium-Positionierung und starker visueller Wiedererkennbarkeit.

## 3. Gewünschte Wirkung/Stimmung
Die Website muss durchgehend folgende Stimmung transportieren: cinematic, feminine, dark, elegant, artsy, modern, mystisch, high-end, editorial.

## 4. Visuelle Grundausrichtung
Die visuelle Leitlinie ist image-first mit einer dunklen Bühne und hochwertiger Editorial-Ästhetik. Bilder, Typografie, Weißraum und Inszenierung sind zentral; UI-Elemente bleiben reduziert und unterstützen die Bildwirkung.

## 5. Referenzprinzipien ohne Kopie
Referenzen dienen ausschließlich als Orientierung für Qualität, Rhythmus und Inszenierung. Eine 1:1-Übernahme bestehender Seiten, Layouts, Komponenten oder markanter Stilmerkmale ist ausgeschlossen. Das Ergebnis muss eigenständig, wiedererkennbar und auf Dama Venus zugeschnitten sein.

## 6. Verbindliche Farbpalette
- Background: `#050505`
- Primary Pink: `#FF4FA8`
- Soft Pink Glow: `#FF8DCC`
- Off-white: `#F3EEF2`
- Muted Text: `#B9B1B7`

Diese Palette ist verbindlich und bildet die Grundlage für Hintergründe, Typografie, Akzente, Overlays und Interaktionszustände.

## 7. Technische Zielbasis
Die Umsetzung basiert verbindlich auf:
- Next.js
- React
- Tailwind CSS
- komponentenbasiertem Aufbau
- responsivem Verhalten über relevante Breakpoints
- performance-orientierter Umsetzung
- mitgedachter Accessibility
- image-first-Ansatz in Struktur und Rendering

## 8. Prompt-/Arbeitslogik
Die Umsetzung erfolgt iterativ in klaren Schritten. Nach jedem Schritt wird die Dokumentation verbindlich fortgeschrieben, sodass Entscheidungen, Änderungen und nächste Maßnahmen jederzeit nachvollziehbar und umsetzungsnah dokumentiert sind.

## 9. Bildstrategie-Hinweis
Echte Bilder werden später im Repository kuratiert und optimiert. Bis dahin wird mit Platzhaltern gearbeitet, die finale Bildlogik, Seitenkomposition und Performance-Anforderungen bereits realistisch vorbereiten.
